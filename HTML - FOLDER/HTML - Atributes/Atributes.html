HTML Attributes

HTML Attributes are used to add more to an HTML element

Important Things to Remember

  • HTML attributes are found in HTML tags
  • HTML attributes only appear at the opening or start tags. it will never be on the end tags
  • HTML element can have multiple attributes
  • HTML element are composed of name/value pairs
  • There are some attribute that can be used on all HTML elements. They are called Global attributes

    • Ab HTML Attribute is composed of the following:

  • an attribute name
  • an equal (=) sign
  • a value surrounded by quotational marks value.
